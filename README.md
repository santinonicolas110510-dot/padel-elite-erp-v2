# Padel Elite ERP

Sistema de Gestión Integral para Clubes de Padel.

## Características
- Gestión de Canchas en tiempo real.
- Control de Caja y Sesiones Diarias.
- Sistema de Señas y Pagos Mixtos.
- Inventario y Facturación POS.
- Estadísticas Avanzadas.

## Tecnologías
- Next.js 15
- Supabase
- Tailwind CSS
- Framer Motion
- Lucide React

## Configuración
Copia el archivo `.env.example` a `.env` y completa las credenciales de Supabase.
